Project Overview
This project features a comprehensive Power BI Dashboard designed to analyze Netflix's vast library of movies and TV shows. By leveraging data visualization, the project identifies trends in content production, global distribution, and genre popularity to provide actionable insights for content strategy.

Key Features & Visualizations

Content Distribution: Breakdown of the library between Movies and TV Shows.


Temporal Trends: Analysis of content additions over the years, highlighting Netflix's growth trajectory.


Geographical Insights: A global map view showing content production by country.
+1


Genre Deep-Dive: Identification of the most popular genres and categories across different regions.


Rating Analysis: Distribution of content ratings (e.g., TV-MA, TV-14) to understand target audience demographics.

Data Source
The dataset used for this analysis contains information on over 8,000 titles available on Netflix, including details like:

Title Type (Movie or TV Show)

Director & Cast

Country of Origin

Date Added to Netflix

Release Year

Rating & Duration

Technical Skills Demonstrated

Data Transformation: Cleaned and modeled data using Power Query (M language) to handle missing values and format dates.


DAX (Data Analysis Expressions): Created custom measures for year-over-year growth and content ratios.


Interactive UI/UX: Implemented slicers, page navigation, and drill-through filters for a dynamic user experience.


Data Storytelling: Structured the layout to guide stakeholders through high-level KPIs down to granular details.

How to View
Download the Netflix_Data_Project.pbix file from this repository.

Open it using Power BI Desktop.

Interact with the filters (Year, Genre, Country) to explore different segments of the data.
